-----------// O Problema //-------------------------------

Colocar 8 rainhas em um tabuleiro de forma 
que nehuma delas esteja sobre ataque


-----------// Criando o tabuleiro //-------------------------------

vamos usar "-" para casas vazias e 
"&" como rainhas

-> Olhe o codigo em "tabuleiro.py"

-// check(i,j) <--- (linha,coluna)  /-
Com o tabuleiro criado vamos criar uma funcao que dado uma rainha ela 
verrifica se esta esta sobre ataque

Note como a funcao ve ambas as diagonais positivas negativas
as colunas e as fileiras...

-> Agora podemos contruir um algoritimo de solucao
  vamos clonar o codigo para manter uma copia intacta da funcao check


-----------// Solucao e algorimo //-------------------------------
Escreva seu algoritimo no arquivo "solucao.py" na parte de solucao
e use a funcao check para checkar se alguma rainha esta sobre ataque
